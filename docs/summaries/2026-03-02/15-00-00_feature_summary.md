# 功能修改总结

## 本次修改的功能点
1. 修改加盟商首页门店经营数据模块的日期显示逻辑
2. 优化报表详情页为移动端适配的二级页面

## 涉及的文件列表
- `App.tsx`：修改报表日期生成逻辑和显示格式
- `components/ReportDetailPage.tsx`：优化移动端适配

## 主要代码逻辑说明
1. **日期显示逻辑修改**：
   - 日报：显示当前日期减1天（如：3-2）
   - 周报：显示为当前日期所在周属于当月的第几周减1周（如：第2周）
   - 月报：显示为当前日期所在月减1月（如：2月）

2. **报表详情页移动端适配**：
   - 减小所有内边距和字体大小
   - 优化卡片圆角和间距
   - 添加点击反馈效果（active:scale-95）
   - 调整图标尺寸以适配小屏幕
   - 优化滚动区域和布局密度

## API 变更记录
无API变更，仅UI和日期逻辑调整

## 功能效果
1. 日报卡片显示昨天的日期（如：03-01）
2. 周报卡片显示上周的周数（如：第2周）
3. 月报卡片显示上个月的月份（如：2月）
4. 报表详情页在移动端显示更紧凑，适合小屏幕浏览
5. 整体布局更加符合移动端交互习惯
