# 功能修改总结

## 本次修改的功能点
修改报表日期显示逻辑：周报显示当前日期所在周减1（上周），月报显示当前日期所在月份减1（上月）。

## 涉及的文件列表
- `App.tsx`：修改 `generateMockReports` 函数中的日期计算逻辑

## 主要代码逻辑说明
1. **周报日期调整**：
   - 原逻辑：`lastWeek.setDate(lastWeek.getDate() - 7)` 直接减去7天
   - 新逻辑：计算当前星期几，然后减去（当前星期几 + 7）天，确保显示上周同一星期几的日期
   - 例如：今天是周三，则显示上周三的日期

2. **月报日期调整**：
   - 原逻辑：已经正确设置为上月
   - 新逻辑：保持不变，`lastMonth.setMonth(lastMonth.getMonth() - 1)`
   - 显示上个月的日期

## API 变更记录
无API变更，仅日期计算逻辑调整

## 功能效果
1. 周报卡片显示上周的日期（如：03-19 如果是本周三03-26）
2. 月报卡片显示上月的日期（如：02-26 如果是本月03-26）
3. 日报保持显示今天的日期
